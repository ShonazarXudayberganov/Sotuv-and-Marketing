# Architecture Decision Records (ADR)

> Muhim arxitekturaviy qarorlar tarixi. Har ADR — bitta qaror.

## Format

```
# ADR-NNNN: Sarlavha

**Holat:** Taklif | Qabul qilingan | Bekor qilingan | O'zgartirilgan
**Sana:** YYYY-MM-DD
**Qaror qabul qilgan:** Tech Lead, Owner, ...

## Kontekst
Nima muammo? Nima uchun qaror kerak?

## Variantlar
1. Variant A — afzalliklar / kamchiliklar
2. Variant B — ...
3. Variant C — ...

## Qaror
Tanlangan variant va sabab.

## Oqibatlar
Ijobiy, salbiy, kelajakda kerak bo'ladigan ishlar.

## Tegishli
- Boshqa ADR'lar
- Tashqi havolalar
```
