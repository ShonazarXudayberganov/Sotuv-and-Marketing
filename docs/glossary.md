# Lug'at (Glossary)

> Loyiha bo'yicha 45+ atama va qisqartmalar.

---

| Atama | Tushuntirish |
|---|---|
| **AI Score** | Mijoz "issiqlik darajasi", 0-100. Qancha katta bo'lsa, sotib olish ehtimoli yuqori. |
| **ARPU** | Average Revenue Per User — bir foydalanuvchidan o'rtacha daromad. |
| **B2B** | Business-to-Business — biznes uchun biznes. |
| **B2C** | Business-to-Consumer — biznes uchun iste'molchi. |
| **CAC** | Customer Acquisition Cost — mijoz jalb qilish narxi. |
| **Churn** | Mijoz ketish foizi. |
| **CTA** | Call To Action — chaqiruv tugma. |
| **CRM** | Customer Relationship Management — mijozlar bilan munosabatni boshqarish. |
| **CTR** | Click-Through Rate — klik / ko'rsatish nisbati. |
| **CPL** | Cost Per Lead — bir lead narxi. |
| **CPC** | Cost Per Click — bir klik narxi. |
| **CPA** | Cost Per Action — bir maqsadli amal narxi. |
| **CPM** | Cost Per Mille — 1000 ko'rsatish narxi. |
| **DAU** | Daily Active Users — kunlik faol foydalanuvchilar. |
| **DM** | Direct Message — shaxsiy xabar (IG/FB). |
| **E2E** | End-to-end test — tizimni butun zanjirda tekshirish. |
| **GDPR** | General Data Protection Regulation — Yevropa shaxsiy ma'lumot qonuni. |
| **HMAC** | Hash-based Message Authentication Code — xabar autentifikatsiyasi. |
| **JWT** | JSON Web Token — auth uchun signed token. |
| **KPI** | Key Performance Indicator — asosiy ko'rsatkich. |
| **KR** | Key Result (OKR'da) — natijali ko'rsatkich. |
| **LTV** | Lifetime Value — mijozdan umumiy daromad. |
| **MAU** | Monthly Active Users — oylik faol foydalanuvchilar. |
| **MCP** | Model Context Protocol — Claude bilan ishlash protokoli. |
| **MQL** | Marketing Qualified Lead — marketingdan kelgan saralangan lead. |
| **MRR** | Monthly Recurring Revenue — oylik takrorlanuvchi daromad. |
| **MVP** | Minimum Viable Product — eng kichik ishlovchi mahsulot. |
| **NPS** | Net Promoter Score — mijoz lojaliylik indeksi (0-10). |
| **OAuth** | Open standard for authentication — uchinchi tomon orqali kirish. |
| **OKR** | Objectives and Key Results — maqsadlar va natijalar tizimi. |
| **PBX** | Private Branch Exchange — ofis telefon tizimi. |
| **pgvector** | PostgreSQL kengaytmasi vector search uchun. |
| **PII** | Personally Identifiable Information — shaxsiy ma'lumot. |
| **RAG** | Retrieval-Augmented Generation — kontekst topib, AI ga berish. |
| **RBAC** | Role-Based Access Control — rolga asoslangan kirish. |
| **ROAS** | Return On Ad Spend — reklama daromadi nisbati. |
| **ROI** | Return On Investment — investitsiya daromadi. |
| **SaaS** | Software as a Service — obuna asosida xizmat. |
| **Schema-per-tenant** | Multi-tenancy strategiya — har mijoz uchun alohida DB schema. |
| **SDK** | Software Development Kit — dasturchilar uchun to'plam. |
| **SLA** | Service Level Agreement — xizmat darajasi kelishuvi. |
| **SMS gateway** | SMS yuborish servisi (Eskiz, Playmobile). |
| **SQL injection** | DB hujum turi — query'ga zaharli kod qo'shish. |
| **SSO** | Single Sign-On — bir martalik kirish. |
| **TDD** | Test-Driven Development — avval test, keyin kod. |
| **Tenant** | Multi-tenancy'da mijoz (kompaniya). |
| **TLS** | Transport Layer Security — kanal himoyasi (HTTPS). |
| **TTL** | Time To Live — saqlash muddati (cache). |
| **Two-way sync** | Ikki tomonlama sinxronizatsiya. |
| **UTM** | Urchin Tracking Module — marketing manba parametrlari. |
| **WAF** | Web Application Firewall — ilova xavfsizlik darvozasi. |
| **WebSocket** | Real-time bidirectional communication. |
| **Webhook** | HTTP callback — hodisa bo'lganda boshqa servisni xabardor qilish. |
| **WSS** | WebSocket Secure (TLS bilan). |
| **Zero-knowledge** | Server foydalanuvchi ma'lumotini ko'rmaydi (encryption foydalanuvchida). |
