# Bosqich 2 — CRM + Inbox

> 3-4 oy. Sotuv va aloqa moduli.

---

## Maqsad

CRM (mijozlar, bitimlar, AI Score, automation, OnlinePBX) va Inbox
(omnikanal, 3 AI engine, sayt widget alohida server).

## KPI

- +150 yangi mijoz (jami 200-250)
- $144K ARR
- Cross-sell: 30%+ existing mijozlar Inbox + CRM oladi
- Inbox auto-respond accuracy > 85%

## Sprintlar

### Sprint 14-15 — CRM asoslari
- Contacts CRUD
- Deals + Pipelines
- Activities timeline
- Mahsulot katalog (SMM bilan sync)

### Sprint 16-17 — CRM AI
- AI Score (auto-adapt)
- Win probability
- Next step recommendations
- Block-based automation builder

### Sprint 18-19 — OnlinePBX va Import
- OnlinePBX integration
- Call recording + transcription (Whisper)
- AmoCRM import wizard

### Sprint 20-21 — Inbox asoslari
- Conversations DB schema
- IG DM, FB Messenger, TG, SMS
- 3-pane UI
- Templates

### Sprint 22-23 — Inbox AI va Site widget
- Sentiment Engine
- Auto-Respond Engine (90% confidence)
- Draft Creator Engine
- **Sayt widget alohida server** (WebSocket cluster)
- SLA monitoring

## Risklar

- **OnlinePBX integratsiyasi murakkab** — soliddan ko'p test
- **Auto-respond xato** — sifatni doimiy monitor
- **Sayt widget scale** — alohida infra dizayni
